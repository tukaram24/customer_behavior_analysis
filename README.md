# 📊 Data Analytics Project README

## 🔎 Overview

This project demonstrates an end-to-end **data analytics workflow**, starting from data loading and exploration to building interactive dashboards and presenting insights. It showcases practical skills in **Python, SQL, and Power BI**, along with data storytelling through reports and presentations.

---

## 📁 Dataset

* The dataset used in this project contains structured data suitable for analysis.
* It includes key attributes required for **data cleaning, transformation, and visualization**.
* Source: Kaggle 

---

## 🛠️ Tools & Technologies

* **Python** – Data loading, cleaning, and exploratory data analysis (EDA)
* **SQL (PostgreSQL / MySQL / SQL Server)** – Querying and data manipulation
* **Power BI** – Dashboard creation and visualization
* **Gamma** – Presentation (PPT) creation
* **Jupyter Notebook** – Development environment

---

## ⚙️ Project Steps

### 1. Data Loading

* Imported dataset using Python libraries such as Pandas.
* Verified structure, columns, and data types.

### 2. Exploratory Data Analysis (EDA)

* Analyzed distributions, trends, and relationships.
* Identified missing values and outliers.
* Used visualizations (charts, graphs) for better understanding.

### 3. Data Cleaning

* Handled missing/null values.
* Removed duplicates and inconsistencies.
* Standardized formats and corrected data types.

### 4. SQL Analysis

* Loaded cleaned data into relational databases:

  * PostgreSQL
  * MySQL
  * SQL Server
* Executed SQL queries for:

  * Data aggregation
  * Filtering and grouping
  * Business insights extraction

### 5. Dashboard Creation

* Built an interactive dashboard in Power BI.
* Included KPIs, charts, and filters for user-driven insights.
* Focused on clarity and usability.

### 6. Reporting & Presentation

* Created a detailed report summarizing findings.
* Designed a professional presentation (PPT) using Gamma.
* Highlighted key insights and recommendations.

## ▶️ How to Run

1. **Clone the Repository**

   ```bash
   git clone https://github.com/tukaram24/customer_behavior_analysis.git
   cd customer_behavior_analysis
   ```

2. **Run Python Scripts / Notebook**

   * Install required libraries:

     ```bash
     pip install -r requirements.txt
     ```
   * Open and run the Jupyter Notebook or Python script.

3. **Set Up Database**

   * Import cleaned dataset into PostgreSQL.
   * Execute SQL queries provided in the `/sql` folder.

4. **Open Power BI Dashboard**

   * Open `.pbix` file in Power BI Desktop.
   * Refresh data connections if needed.

5. **View Presentation**

   * Open Gamma PPT or exported presentation file.
